    1.武林送货     (送货物到某个城市的某人) 直接give后完成任务
    ->宝物争夺 或 寻找物品
    (give)
    
    
    2.边疆御敌     (到了指定地点 citan出现3-5个敌人) 杀了有奖励
    ->门派守护 或 江湖追杀
    (npc)
    
    3.门派守护     (到了指定地点 citan出现3-5个敌人) 杀了有奖励
    ->边疆御敌 或 江湖追杀
    (npc)
    
    4.江湖追杀     (到指定地点附近 找到敌人  杀之)
    ->武林送货 或 情报刺探
    (npc)
    
    5.保镖护送     (护送一个武林人物 到达目的地，先ask npc about start 他就会跟随你了。ask npc about over)
    ->武林送货 或 江湖追杀
    (npc)
    
    6.宝物争夺     (找到相关地点的敌人，杀之，得到身上的物品，give 任务发布人)
    ->保镖护送 或 人物探寻
    (give)
    
    7.人物探寻     (到某地找到相关人，ask npc about start 然后把他带到 任务发布人边上 ask npc about over)
    ->宝物争夺 或 寻找物品
    (ask)
    
    8.寻找物品     (找到某物 直接给任务发布人)
    ->情报刺探 或 保镖护送
    (give)
    
    9.情报刺探     (找到某地 刺探任务(citan))
    ->门派守护 或 边疆御敌
    (citan)
    
    连续5次->出现boss挑战citan (6次完成任务)
